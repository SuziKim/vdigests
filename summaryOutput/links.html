Open one link at a time using Chrome. Allow 5 seconds for the app to load the summary. Clicking on a thumbnail will start the video player at the begining of the summarized section.

If you open more than one link for the same video at once, the videos will not load correctly. This is because Chrome does not load videos properly when multiple players are accessing a single file. (On a single summary page, each player accesses a different video file.)
